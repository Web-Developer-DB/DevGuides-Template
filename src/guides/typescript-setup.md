# TypeScript Setup in React (Bestandsprojekt)

```bash
npm i -D typescript @types/react @types/react-dom
npx tsc --init
```

- .jsx → .tsx umbenennen
- `import React from 'react'` sicherstellen, falls nötig

## Checkliste
- `typescript` und React-Typen installiert.
- `tsconfig.json` erzeugt.
- `.jsx` Dateien nach `.tsx` umbenannt.
- Build und Tests laufen ohne TS-Fehler.
