# DevGuides – React Router Starter

Ein erweiterbarer Starter für **React + React Router (v6)** mit responsivem Layout und Dokumentationsstruktur für Anleitungen & Codebeispiele.

## Features
- Vite + React 18 + React Router 6
- Tailwind CSS inkl. Typography-Plugin
- Responsives Layout mit Navbar, Footer und Mobile-Hamburger
- Guides-Übersicht mit Suche (⌘/Ctrl+K) sowie Themen-Navigation
- Beispiel-Guides unter `docs/examples/`
- Theme-Switcher mit Dracula-Dark-Theme und persistenter Auswahl
- ESLint + Prettier für konsistente Codequalität

## Schnellstart
```bash
npm install
npm run dev
```

Weitere Details findest du in `docs/README-DEV.md`.

## Quality-Checks
- `npm run lint` / `npm run lint:fix`
- `npm run format:check` / `npm run format`
- `npm test` / `npm run test:watch`
