# DevGuides Template – Entwickler-Info

## Lokale Entwicklung
- `npm run dev` – startet Vite Dev-Server
- `npm run build` – Produktion-Build
- `npm run preview` – Build lokal testen
- `npm run lint` / `npm run lint:fix` – Code-Qualität & Autofix
- `npm run format:check` / `npm run format` – Formatierung mit Prettier
- `npm test` / `npm run test:watch` – Vitest + React Testing Library

## Architektur
- **pages/**: Routen-Ansichten
- **components/**: wiederverwendbare UI
- **data/**: Metadaten deiner Guides (Titel, Tags, Summary)
- **docs/**: Redaktions-Dokumente & Beispiel-Guides

## Deployment
- Static Hosting (z. B. Netlify, Vercel, GitHub Pages)
